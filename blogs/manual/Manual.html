    • 安装node环境

      新npm设置淘宝源 - 知乎 (zhihu.com) (opens new window)

    # 使用说明

    项目下载到本地后，本地需安装****node环境*及设置好*npm仓库源****（避免依赖下载失败）。

    node安装 (opens new window)

    npm仓库源设置 (opens new window)

    执行以下命令

    1. 下载依赖

    npm i

    1. 运行项目

    npm run dev
